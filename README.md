## 🔥 Функциональные возможности

* **Multi-Factor Scoring:** Анализ акций по набору индикаторов (RSI, Moving Averages, Volume Spikes).
* **AI Interpretation:** Использование Claude API для анализа новостного фона и синтеза технического анализа в человекочитаемый отчет.
* **Portfolio Tracker:** Слежение за выбранными активами и автоматическое уведомление о достижении зон интереса.
* **Historical Backtesting:** Модуль проверки эффективности стратегии отскока на исторических данных.
* **Structured Reporting:** Формирование кратких и подробных сводок по рынку прямо в мессенджер.

## 🚀 Архитектура системы

Проект построен по модульному принципу, что позволяет легко масштабировать его под любые финансовые инструменты:

* `analysis/` — Ядро технического анализа и расчета индикаторов.
* `scoring/` — Алгоритм ранжирования акций по стратегии "Bounce".
* `llm/` — Логика взаимодействия с ИИ для генерации финальных выводов.
* `backtest/` — Среда для тестирования гипотез на прошлых периодах.
* `bot/` — Асинхронный интерфейс взаимодействия с пользователем.
